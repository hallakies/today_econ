# 오늘경제 편집 대원칙

오늘경제의 최종 결과물(사진과 글)은 아래 두 감정을 만들어야 한다.

1. “와! 이 계정 하나 구독하면 중요한 경제 소식을 놓치지 않고 알 수 있어!”
2. “와! 여기는 가독성이 좋고, 쉽게 정보를 알려줘.”

## 운영 규칙

- 중요한 뉴스의 기준은 독자의 돈에 직접 미치는 영향이다. 주식·부동산·생활물가·대출·세금을 모두 포함한다.
- 정책 뉴스는 먼저 집값·대출·세금에 생기는 구체적 변화를 보여준다. 정책 주체의 반박과 논쟁은 그 다음에 근거와 함께 비교한다.
- 모든 설명은 “무엇이 바뀌었고 → 그래서 내 돈의 무엇이 달라지는가”로 연결한다.
- 독자에게 보이는 기본 라벨은 `무슨 일이야?` → `그래서 내 돈은?` → `오늘경제 한 줄 생각` → `앞으로 이렇게 될 수도`를 사용한다.
- 기사에 없는 영향 분석은 가능성으로 말할 수 있지만, 확인 사실과 섞지 않는다. `~할 수 있어요`, `~일 가능성이 있어요` 같은 표현을 쓴다.
- `undefined`, 비문, 문장 파편, 근거 없는 숫자·시행일은 게시하지 않는다.
- 좋아요·팔로우를 위해 공포, 정치적 편향, 매수·매도 권유를 사용하지 않는다.

## 첫 4주 최소 성공선

게시 성공률 100%, `undefined`·비문 0건, 게시물마다 팔로우·저장·공유 중 하나 이상 발생을 기준으로 삼는다. 이후 24시간·72시간 데이터를 쌓아 어떤 주제와 설명 방식이 독자의 돈 이해와 행동을 높이는지 측정한다.
